Что нового в TheQube

На этой странице представлены все изменения и исправления в TheQube, от более новых к более старым.

TheQube 0.8

Изменения

  • +Добавлено: Добавлена поддержка длинных личных сообщений (до 10000 символов).
  • +Добавлено: добавлена поддержка цитирования твитов. Чтобы процитировать твит, нажмите Ctrl+Shift+Win+Q. Если вы читаете твит с цитатой, процитированный твит будет предваряться буквами QT (по аналогии с RT для ретвитов). Благодарим Мануэля Кортеса за замечательную помощь!
  • +Добавлено: Добавлен сервис сокращения ссылок ux.nu.
  • *Изменено: Улучшена технология хранения данных учётных записей Твиттера.
  • *Изменено: изменены некоторые комбинации клавиш для совместимости с Windows 10:
    • Перемещение по элементам в буфере изменено на Alt+Win+Стрелки вверх и Вниз;
    • Перемещение к следующему и предыдущему буферу изменено на Alt+Win+стрелки вправо и влево;
    • Диалог нового личного сообщения вызывается по Alt+Win+D;
    • Если вы используете SAPI 5, останавливайте речь по Alt+Win+S;
    • Закрывайте TheQube по Ctrl+Alt+Win+F4.
  • Избранное и всё, связанное с ним, переименовано в «нравится» и т.д. в соответствии с требованиями Твиттера.
  • Документация сконвертирована в написанный вручную HTML 5 с разметкой WAI-ARIA. Перемещайтесь не только по заголовкам, но и по областям!

Исправления

  • #49: Для авторизации второй сессии требуется перезапуск TheQube. Благодарим Мануэля Кортеса за это замечательное исправление!
  • #56: При чтении, прекращении чтения и блокировании TheQube не проверяет на повтор действия.
  • #48: Пропадание переводов строки при чтении твита.
  • #25: Невозможно вставить перевод строки, если включена отправка твитов по клавише Enter.
  • #16: Ошибки при завершении работы Windows.
  • #62: Невозможно получить оставшееся число вызовов API. Благодарим за это исправление Крейга Бретта.
  • -Исправлено: Переводчик Google снова с нами.
  • -Исправлено: Ошибки в переводе на португальский (Бразилия), которые мешали TheQube нормально функционировать.

TheQube 0.7

Изменения

  • Теперь Программа называется TheQube (без пробела) и подчиняется семантической системе нумерации версий версия.функция.исправление. До выхода первой стабильной версии первой цифрой номера будет нуль, поэтому данная версия имеет номер 0.7.
  • Вкладки «Речь» и «Брайль» в глобальной конфигурации программы объединены в одну, так как вкладка «Брайль» содержала только один флажок.
  • Служба перевода твитов снова работает (#27)
  • Теперь поддерживаются аудиоссылки с изменённого сервиса AudioBoom.
  • Теперь у TheQube есть постоянный web-сайт
  • Теперь адрес web-сайта в диалоге информации о профиле не будет сокращаться с помощью t.co.
  • Теперь в диалоге информации о профиле будет отображаться также число пользователей, добавивших текущего пользователя в свои списки.
  • Папка doc удалена из исходного кода, так как не содержит значимого руководства для разработчиков TheQube. Если штат разработчиков будет увеличен, такое руководство будет написано заново.
  • Временно удалена поддержка сервиса сокращения ссылок hkc.im из-за ошибки сервиса, см. их главную страницу. Поддержку можно вернуть в любой момент, как только сервис заработает снова.
  • Добавлена поддержка новых сервисов сокращения ссылок, а именно: goo.gl, senta.la и readability.
  • При авторизации ввод пин-кода больше не требуется. Однако окно браузера всё же открывается, пока с этим сделать ничего нельзя (#13)
  • Обновлена локализация на испанский, сербский и польский языки. Спасибо нашей команде переводчиков!

Исправления

  • -Исправлено: #33: Переключение сессий прекращает работу клавиатурных команд
  • -Исправлено: #1: Клавиатурные команды не работают, если по умолчанию раскладка клавиатуры не установлена на английский язык
  • -Исправлено: #44: Язык документации TheQube не соответствует выбранному при установке

The Qube 1.0 beta 6

Изменения

  • Сочетание клавиш для обновления буфера изменено на Ctrl+Win+Alt+U, чтобы разрешить конфликт с клавиатурными командами Windows 8.
  • Исходный код The Qube перемещён на Github.
  • Теперь у The Qube есть багтрекер — место, где можно сообщить о проблеме или предложить нововведение. Он находится здесь. Сообщения лучше оставлять на английском языке. Если вы не владеете английским, пожалуйста, не используйте машинный перевод — пишите по-русски.
  • Теперь вы можете решить, отправлять сообщение по клавише Enter или по Ctrl+Enter. Соответствующий флажок находится в конфигурации The Qube, на вкладке «Разное». По умолчанию сообщения отправляются по клавише Enter, как и раньше.
  • Теперь вы можете выбрать, следует ли загружать прикрепляемое аудио на SndUp или на TwUp.
  • Сессия GoogleVoice удалена ввиду невозможности её тестирования разработчиками (сервис работает только в США).
  • Теперь геолокация сообщается на том языке, который вы установили для интерфейса The Qube.
  • Сочетания клавиш для начала/прекращения чтения пользователя изменены на Ctrl+Win+S и Ctrl+Shift+Win+S для разрешения конфликта с функцией переключения языка в JAWS 16. Чтобы запомнить новые сочетания, используйте слова «Start/stop following».

Исправления

  • Теперь вы можете отправлять сообщения, содержащие переводы строки. По умолчанию перевод строки вставляется с помощью Ctrl+Enter. Однако вы можете пожелать отправлять сообщения по Ctrl+Enter, тогда перевод строки будет вставляться по клавише Enter.
  • Редактирование сочетаний клавиш снова работает.
  • Теперь вы можете выбрать голос SAPI5-синтезатора, если используете его для The Qube.
  • Информация о геолокации снова сообщается всеми способами (Ctrl+Win+G, Ctrl+Shift+Win+G, Ctrl+Shift+Win+Alt+G).
